# SpendDNA: Wallet Intelligence Pipeline 📊

A personal finance analytics tool that transforms raw bank statements into an actionable "Wrapped" report, highlighting spending patterns, monthly trends, and financial anomalies.

## 🚀 Overview
SpendDNA is designed to provide visibility into discretionary spending. It categorizes hundreds of transactions automatically, identifies late-night habits, and flags anomalous spending using Z-score analysis.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy
- **Environment:** Google Colab / Jupyter Notebook

## 📈 Key Features
* **Automated Parsing:** Handles multi-format date parsing and messy bank descriptions.
* **Smart Categorization:** Uses a custom mapping engine to tag 12+ categories.
